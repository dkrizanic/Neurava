# Epic 2 Context: Core Notebook Experience

## Goal

Users can create, edit, organize, search, and manage notes in a polished workspace-scoped notebook UI.

## Stories

- Story 2.1: Create And View Workspace-Scoped Notes
- Story 2.2: Edit Notes With Autosave Feedback
- Story 2.3: Archive And Restore Notes
- Story 2.4: Organize Notes With Tags, Favorites, And Links
- Story 2.5: Manual Note Search And Filters
- Story 2.6: Selectable Professional Editor Modes

## Implementation Notes

- Notes are scoped by active Workspace Context.
- Notes support title, body, dates, archive/restore, tags, favorite/pinned flags, linked resources, and editor mode.
- Manual search and filters operate without AI.
- The notes UI includes loading, empty, error, create, edit, archive, restore, and organization states.
